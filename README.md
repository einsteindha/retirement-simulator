# 은퇴 인출 시뮬레이터
**정원준 CFP** 재무상담용 은퇴 인출 시뮬레이터

🔗 **라이브 데모**: `https://einsteindha.github.io/retirement-calculator/`

---

## 주요 기능

### 계산 엔진
- **몬테카를로 시뮬레이션** 10,000회 — 로그정규분포 기반으로 실제 시장 수익률 특성 반영
- **올바른 복리 공식** — 현재 자산 × (1+수익률)^기간 + 저축액 연금미래가치
- **적립기 / 인출기 수익률 분리** — 자산 성장 추산과 몬테카를로에 각각 다른 수익률 적용
- **시점 기반 현금 버퍼** — 은퇴·배우자은퇴·연금개시 시점으로 구간을 나눠 정확한 버퍼 산출

### 시나리오
- **시나리오 A** — 부부/동반 함께 생존 (기준)
- **독거 시나리오** — 기대수명 기준 자동 결정 (수명 짧은 쪽이 먼저 사망)
- **수익률 순서 시나리오** — 특정 구간에 고정 수익률 적용 시나리오 비교 (시퀀스 리스크 시각화)

### 입력 섹션 (12개)

| 섹션 | 내용 |
|------|------|
| 1 · 본인 정보 | 나이·은퇴나이·은퇴준비자금·저축 |
| 2 · 배우자 정보 | 배우자 나이·은퇴나이·저축 (토글) |
| 3 · 기대수명 설정 | 본인·배우자 기대수명 → 시나리오별 기간 자동 계산 |
| 4 · 은퇴 자산 포트폴리오 | 적립기/인출기 기대수익률·변동성 직접 입력 |
| 5 · 공적연금 (부부) | 조기·정상·연기 전략, 손익분기 자동 계산 |
| 6 · 독거자 공적연금 입력 | 독거자 연금·유족연금·기타 직접 입력 |
| 7 · 인생 단계별 생활비 설계 | 활동기·회상기·간병기·독거활동기·독거간병기 |
| 8 · 자산 유출 | 일시금 또는 정기금 지출 (자녀 독립자금 등), 물가 자동 반영 |
| 9 · 일시 자산 유입 | 상속·증여·부동산 매각 등 |
| 10 · 추가 현금흐름 수입 | 임대소득·브릿지잡 등 물가연동 정기 수입 |
| 11 · 정액 수령 연금 | 주택연금·보험사 연금 등 명목 고정 수입 |
| 12 · 기타 인출 조건 | 현금 버퍼 기간·물가상승률·상속 목표 (0~100억) |

## 포트폴리오 수익률 입력 방법

백테스트 도구로 수익률·변동성을 확인 후 입력하세요.

**추천 도구**
- [IdentiFi 자산배분 백테스트](https://identifi.co.kr) — 수익률·변동성 확인
- CFP 상담 결과값 직접 입력

**입력 가이드**
- **적립기**: 은퇴 전 자산 성장 추산 (공격적, 예: 7~9%)
- **인출기**: 몬테카를로 시뮬레이션 (보수적, 예: 4~6%)

## 계산 방식

| 항목 | 방식 |
|------|------|
| 은퇴 시점 자산 | `현재자산 × (1+r)^n + 저축액 × ((1+r)^n-1)/r` |
| 인출 필요액 | `(월생활비 - 공적연금) × 12 × (1+인플레)^y` |
| 정액연금 차감 | 명목 고정 (물가 연동 없음) |
| 현금흐름 차감 | 물가 연동 (실질가치 유지) |
| 몬테카를로 수익률 | 로그정규분포: `exp(μ + σ×Z) - 1`, Z~N(0,1) |
| 현금 버퍼 | 시점 기반 구간별 인출 필요액 × 버퍼 기간 |
| 생존 확률 | 10,000가지 시나리오 중 자산 고갈 없이 상속 목표 달성 비율 |
