# 은퇴 안전 인출 계산기

**정원준 CFP · 서울리(Re)테크** 재무강의용 은퇴 인출 시뮬레이터

🔗 **라이브 데모**: `https://[your-username].github.io/retirement-calculator/`

---

## 주요 기능

- **몬테카를로 시뮬레이션** (1,200회) 기반 자산 생존 확률 계산
- **3가지 시나리오** 비교
  - A: 부부 함께 생존 (기준)
  - B: 본인 먼저 사망 → 배우자 홀로
  - C: 배우자 먼저 사망 → 본인 홀로
- **인생 단계별 생활비** 설계 (소득공백기 · 활동기 · 회상기 · 간병기 · 독거기)
- **이벤트 비용** 추가 입력 (자녀 독립자금 등)
- **공적연금 수령 전략** (조기 / 정상 / 연기) 손익분기 계산
- **독거 시나리오 공적연금 직접 입력** (유족연금 등)
- **포트폴리오 입력** — 간편 (수익률·변동성 직접) / 상세 (자산군별)
- 다크모드 자동 지원

## 계산 방식

| 항목 | 방식 |
|------|------|
| 인출 필요액 | `(월 생활비 - 공적연금) × 12 × (1 + 인플레)^y` |
| 자산 성장 | 몬테카를로: `ret ± vol × 1.5 × 난수` |
| 은퇴 시점 자산 | `현재 자산 × (1 + ret×0.6)^기간 + 저축 × 기간` |
| 이벤트 비용 | 현재 물가 기준 입력 → `× (1 + 인플레)^발생연도` |
| 생존 확률 | 1,200회 시뮬레이션 중 자산 고갈 없이 상속 목표 달성 비율 |
